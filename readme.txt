== Installation ==

=== Manual ===
1. Upload the Almond Stock Market Data folder to the plugins directory in your WordPress installation
2. Activate the plugin.
3. Navigate to the "Widget" Menu under Appearance or clicking in "Settings" below the plugin name once activated.

That's it! Now you can easily fill the Tickers below with the tickers of the company you want to show in your website, for instance if you write 'ko' you will get 'Coca-Cola Company (The)' information.
