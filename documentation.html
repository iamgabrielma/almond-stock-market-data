A) Installation - top

Once you have your zip file from Envato you only need to upload it through the administration area in your WordPress website. Click on Dashboard > Plugins > Add New.

In the Add Plugins page you only have to click on the "Upload Plugin" button and select the file containing your new plugin.

Once you click the Install now button WordPress will take care of the process and show you a new page similar to this one

The plugin will appear in your Installed Plugins list, and once you activate it you will see few options under its name :

  • Deactivate: You can deactivate or activate Almond Stock Market Data clicking here.
  • Edit: This link will redirect you to the source code of the plugin. I do not recommended to change anything from the source code if you do not fully understand what are you doing. In any case I'm available for customizations, just contact me if you need to customize Almond Stock Market Data to fit your needs.
  • Settings: Clicking here will redirect you to the Widgets page, where the plugin can be used and configured.
  • More plugins by AlmondWP: More plugins created by me :)

The next step will be clicking on Settings so you can use the plugin in your website

B) Using Almond Stock Market Data - top

Activate it under Appearance > Widgets > Available Widgets or clicking in Settings below the plugin name as shown in the Installation guide. Once you're in the Widgets page you only need to grab the Almond Stock Market Data widget from the Available Widgets column and drag and drop it into the widget are you like (primary, secondary, footer, homepage, ... options will depend on your theme).

  • Title: The title you'd like to show in the front of your website in the top of the widget, if any.
  • Tickers: a ticker is the one to four letter code that represents a particular stock or company in the stock market, writing the ticker of any company available will return the actual price and volume about the company.
    • Examples:
    • The Coca-Cola Co -> ticker: KO
    • Procter & Gamble Co -> ticker: PG
    • Amazon.com, Inc. -> ticker: AMZ
    • AT&T Inc. -> ticker: T

C) Configuration options - top

  • Activate links on tickers: This option will activate create a link to Yahoo Finances on the ticker, so when the ticker is clicked it will open a new window in your browser that will show you all the information regarding that company.
  • Use Google Finances: The default previous link will redirect you to the company information on Yahoo Finances, clicking here it will open a new window in your browser that will show you all the information regarding the company in Google Finances instead.
  • Show Trading Volumes: Activating this option the trading volume for the stocks will appear also in your website, not only the price, name and ticker of the company.
